HTML Tags Ordered Alphabetically

Tags Description
!-- -- Tag Defines a comment
!DOCTYPE Tag Defines the document type
a Tag Defines a hyperlink
abbr Tag Defines an abbreviation or an acronym
acronym Tag Not supported in HTML5. Use abbr Tag instead.Defines an acronym
address Tag Defines contact information for the author/owner of a document
applet Tag Not supported in HTML5. Use embed Tag or object Tag instead. Defines an embedded applet
area Tag Defines an area inside an image map
article Tag Defines an article
aside Tag Defines content aside from the page content
audio Tag Defines embedded sound content
b Tag Defines bold text
base Tag Specifies the base URL/target for all relative URLs in a document
basefont Tag Not supported in HTML5. Use CSS instead. Specifies a default color, size, and font for all text in a document
bdi Tag Isolates a part of text that might be formatted in a different direction from other text outside it
bdo Tag Overrides the current text direction
big Tag Not supported in HTML5. Use CSS instead. Defines big text
blockquote Tag Defines a section that is quoted from another source
body Tag Defines the document's body
br Tag Defines a single line break
button Tag Defines a clickable button
canvas Tag Used to draw graphics, on the fly, via scripting (usually JavaScript)
caption Tag Defines a table caption
center Tag Not supported in HTML5. Use CSS instead.Defines centered text
cite Tag Defines the title of a work
code Tag Defines a piece of computer code
col Tag Specifies column properties for each column within a colgroup Tag element
colgroup Tag Specifies a group of one or more columns in a table for formatting
data Tag Adds a machine-readable translation of a given content
datalist Tag Specifies a list of pre-defined options for input controls
dd Tag Defines a description/value of a term in a description list
del Tag Defines text that has been deleted from a document
thead Tag Groups the header content in a table
time Tag Defines a specific time (or datetime)
title Tag Defines a title for the document
tr Tag Defines a row in a table
track Tag Defines text tracks for media elements (video Tag and audio Tag)
tt Tag Not supported in HTML5. Use CSS instead. Defines teletype text
u Tag Defines some text that is unarticulated and styled differently from normal text
ul Tag Defines an unordered list
var Tag Defines a variable
video Tag Defines embedded video content
wbr Defines a possible line-break
